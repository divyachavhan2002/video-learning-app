# 🎓 Video Learning App

A modern video learning platform built with Next.js and Firebase - Internship Project.

## 🚀 Features

- 📹 Video course platform
- 👤 User authentication (Login/Signup)
- 📚 Course enrollment system
- 📊 Progress tracking
- 🎯 User dashboard
- 📱 Responsive design

## 🛠️ Tech Stack

- **Frontend:** Next.js (Pages Router), React
- **Backend:** Firebase (Authentication + Firestore Database)
- **Video Player:** React Player
- **Styling:** CSS Modules
- **Deployment:** Vercel

## 📦 Installation

1. **Clone the repository:**
```bash
git clone https://github.com/divyachavhan2002/video-learning-app.git
cd video-learning-app
```

2. **Install dependencies:**
```bash
npm install
```

3. **Run the development server:**
```bash
npm run dev
```

4. **Open in browser:**
```
http://localhost:3000
```
